# Android App Automation using Droidrun

## Problem
Manual interaction with Android apps is time-consuming and repetitive.

## Solution
This project demonstrates a simple Android automation agent built using the Droidrun framework.
The agent automatically opens an app and performs basic UI actions.

## Features
- Launches an Android app automatically
- Performs a simple UI interaction
- Demonstrates basic automation capability

## Tech Stack
- Droidrun Framework
- Android Emulator / Android Device
- Java / Kotlin (basic)
